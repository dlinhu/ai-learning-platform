# Ambiguity & Clarity / 消歧与清晰度

## Overview / 概述

Learn techniques for creating clear and unambiguous prompts. / 学习创建清晰无歧义提示的技术。

## Key Knowledge Points / 核心知识点

### 1. Ambiguity Sources / 歧义来源

**English:** Ambiguity sources include lexical ambiguity, syntactic ambiguity, semantic ambiguity, and contextual ambiguity that affect prompt understanding.

**中文:** 歧义来源包括词汇歧义、句法歧义、语义歧义、上下文歧义等影响提示理解的因素。

**Key Concepts / 核心概念:**
- Lexical ambiguity / 词汇歧义
- Syntactic ambiguity / 句法歧义
- Contextual ambiguity / 上下文歧义

**Example / 示例:**
```
The chicken is ready to eat. (Who is eating?)
→ Clarify: The chicken is cooked and ready to be eaten.

```

---

### 2. Clarity Techniques / 清晰度技术

**English:** Clarity techniques include using specific vocabulary, clarifying context, providing examples, and defining terms to eliminate ambiguity.

**中文:** 清晰度技术包括使用具体词汇、明确上下文、提供示例、定义术语等方法消除歧义。

**Key Concepts / 核心概念:**
- Specific vocabulary / 具体词汇
- Clear context / 明确上下文
- Explicit definitions / 明确定义

**Example / 示例:**
```
Instead of 'Process the data', say 'Extract customer names from the CSV file and save to a new column.'
→ Unambiguous instruction

```
