# Ethical Prompt Engineering / 伦理提示工程

## Overview / 概述

Learn ethical considerations and best practices in prompt engineering. / 学习提示工程中的伦理考量和最佳实践。

## Key Knowledge Points / 核心知识点

### 1. Ethical Principles / 伦理原则

**English:** Ethical principles include fairness, transparency, privacy protection, and harm avoidance as basic guidelines for prompt design.

**中文:** 伦理原则包括公平性、透明性、隐私保护、避免伤害等指导提示设计的基本准则。

**Key Concepts / 核心概念:**
- Fairness / 公平性
- Transparency / 透明性
- Privacy protection / 隐私保护
- Harm avoidance / 避免伤害

**Example / 示例:**
```
Evaluate candidates based on skills and experience only, without considering gender, age, or ethnicity.
→ Bias-free evaluation criteria

```

---

### 2. Bias Mitigation / 偏见缓解

**English:** Bias mitigation involves identifying and reducing bias in prompts to ensure AI outputs treat all groups fairly.

**中文:** 偏见缓解涉及识别和减少提示中的偏见，确保AI输出公平对待所有群体。

**Key Concepts / 核心概念:**
- Bias identification / 偏见识别
- Fair representation / 公平代表
- Inclusive language / 包容性语言

**Example / 示例:**
```
Use gender-neutral language: 'they' instead of 'he/she', 'chairperson' instead of 'chairman'.
→ Inclusive output language

```
