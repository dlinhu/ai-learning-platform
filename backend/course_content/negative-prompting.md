# Negative Prompting / 负面提示

## Overview / 概述

Learn negative prompting techniques for specifying what to avoid in AI outputs. / 学习负面提示技术，指定AI输出中要避免的内容。

## Key Knowledge Points / 核心知识点

### 1. Negative Constraints / 负面约束

**English:** Negative constraints guide models to avoid specific outputs by explicitly stating what should not be included.

**中文:** 负面约束通过明确指出不应包含的内容，引导模型避免特定输出。

**Key Concepts / 核心概念:**
- Exclusion rules / 排除规则
- Avoidance instructions / 避免指令
- Negative examples / 负面示例

**Example / 示例:**
```
Write a product description. Do NOT use:
- Superlatives (best, greatest)
- Exaggerated claims
- Technical jargon
→ Grounded, accessible description

```

---

### 2. Use Cases / 使用场景

**English:** Negative prompting is suitable for content filtering, style control, safety restrictions, and other scenarios requiring exclusion of specific outputs.

**中文:** 负面提示适用于内容过滤、风格控制、安全限制等需要排除特定输出的场景。

**Key Concepts / 核心概念:**
- Content filtering / 内容过滤
- Style control / 风格控制
- Safety restrictions / 安全限制

**Example / 示例:**
```
Generate a story for children. Do NOT include:
- Violence
- Scary elements
- Complex vocabulary
→ Child-friendly story

```
