# Evaluating Prompt Effectiveness / 提示效果评估

## Overview / 概述

Learn methods for evaluating and measuring prompt effectiveness. / 学习评估和测量提示效果的方法。

## Key Knowledge Points / 核心知识点

### 1. Evaluation Metrics / 评估指标

**English:** Evaluation metrics include accuracy, consistency, relevance, and completeness as standards for quantifying prompt effectiveness.

**中文:** 评估指标包括准确性、一致性、相关性、完整性等用于量化提示效果的标准。

**Key Concepts / 核心概念:**
- Accuracy / 准确性
- Consistency / 一致性
- Relevance / 相关性
- Completeness / 完整性

**Example / 示例:**
```
Accuracy = (Correct outputs / Total outputs) × 100%
→ Quantitative evaluation measure

```

---

### 2. Evaluation Methods / 评估方法

**English:** Evaluation methods include human evaluation, automated evaluation, A/B testing, and user feedback as ways to collect and analyze effectiveness data.

**中文:** 评估方法包括人工评估、自动评估、A/B测试、用户反馈等收集和分析效果数据的方式。

**Key Concepts / 核心概念:**
- Human evaluation / 人工评估
- Automated evaluation / 自动评估
- A/B testing / A/B测试

**Example / 示例:**
```
1. Define test cases
2. Run prompts
3. Collect outputs
4. Score against criteria
5. Calculate metrics
→ Systematic evaluation workflow

```
