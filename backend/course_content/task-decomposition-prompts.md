# Task Decomposition Prompts / 任务分解提示

## Overview / 概述

Learn task decomposition techniques that break complex problems into manageable sub-tasks. / 学习任务分解技术，将复杂问题分解为可管理的子任务。

## Key Knowledge Points / 核心知识点

### 1. Decomposition Strategies / 分解策略

**English:** Decomposition strategies include functional decomposition, temporal decomposition, and complexity-based decomposition to break large tasks into manageable sub-tasks.

**中文:** 分解策略包括按功能分解、按时间分解、按复杂度分解等方法，将大任务拆分为可管理的小任务。

**Key Concepts / 核心概念:**
- Functional decomposition / 功能分解
- Hierarchical structure / 层次结构
- Manageable sub-tasks / 可管理子任务

**Example / 示例:**
```
Break down 'Launch a product' into:
1. Research
2. Development
3. Testing
4. Marketing
→ Structured task breakdown

```

---

### 2. Decomposition Prompts / 分解提示

**English:** Decomposition prompts are specially designed prompt templates to guide models in task decomposition, including decomposition rules and output formats.

**中文:** 分解提示是专门设计用于引导模型进行任务分解的提示模板，包含分解规则和输出格式。

**Key Concepts / 核心概念:**
- Decomposition rules / 分解规则
- Output format / 输出格式
- Template design / 模板设计

**Example / 示例:**
```
Decompose the following task into sub-tasks:
- Each sub-task should be completable in one step
- List dependencies between sub-tasks
Task: {task}
→ Ordered sub-task list with dependencies

```
