# Chain of Thought (CoT) Prompting / 思维链提示

## Overview / 概述

Learn Chain of Thought prompting that encourages AI models to break down complex problems into step-by-step reasoning. / 学习思维链提示技术，让AI模型将复杂问题分解为逐步推理。

## Key Knowledge Points / 核心知识点

### 1. Basic CoT Prompting / 基础思维链

**English:** Basic CoT prompting adds trigger phrases like 'Let's think step by step' to make models show reasoning processes instead of giving direct answers.

**中文:** 基础思维链提示通过添加'让我们一步一步思考'等触发语，让模型展示推理过程而非直接给出答案。

**Key Concepts / 核心概念:**
- Step-by-step reasoning / 逐步推理
- Reasoning visibility / 推理可见性
- Trigger phrases / 触发语

**Example / 示例:**
```
Solve step by step:
If a train travels 120 km in 2 hours, what is its average speed?
→ Step 1: Speed = Distance / Time
Step 2: 120 / 2 = 60 km/h

```

---

### 2. Advanced CoT Techniques / 高级思维链技术

**English:** Advanced CoT techniques include multi-step reasoning, self-consistency checks, and other strategies to further improve complex problem-solving.

**中文:** 高级思维链技术包括多步推理、自我一致性检查等策略，进一步提高复杂问题的解决能力。

**Key Concepts / 核心概念:**
- Multi-step analysis / 多步分析
- Self-verification / 自我验证
- Complex reasoning / 复杂推理

**Example / 示例:**
```
For each step:
1. State what to calculate
2. Write the formula
3. Perform calculation
4. Explain result
→ Structured reasoning output

```

---

### 3. CoT vs Standard Prompting / 思维链与标准提示对比

**English:** Compared to standard prompting, CoT significantly improves accuracy and interpretability on complex tasks like math problems and logical reasoning.

**中文:** 思维链提示相比标准提示，在数学问题、逻辑推理等复杂任务上显著提高准确性和可解释性。

**Key Concepts / 核心概念:**
- Accuracy improvement / 准确性提升
- Interpretability / 可解释性
- Complex task benefit / 复杂任务优势

**Example / 示例:**
```
Standard: 'What is the answer?'
CoT: 'Let's solve this step by step...'
→ CoT shows reasoning process

```
