# Prompt Formatting & Structure / 提示格式与结构

## Overview / 概述

Learn prompt formatting and structuring techniques for clearer and more effective prompts. / 学习提示格式化和结构化技术，创建更清晰有效的提示。

## Key Knowledge Points / 核心知识点

### 1. Formatting Principles / 格式化原则

**English:** Formatting principles include using clear delimiters, consistent formatting, appropriate indentation and line breaks to improve prompt readability.

**中文:** 格式化原则包括使用清晰的分隔符、一致的格式、适当的缩进和换行，提高提示的可读性。

**Key Concepts / 核心概念:**
- Delimiters / 分隔符
- Consistency / 一致性
- Readability / 可读性

**Example / 示例:**
```
### Task ###
{task}

### Context ###
{context}

### Output Format ###
{format}
→ Clearly structured prompt

```

---

### 2. Structural Elements / 结构元素

**English:** Structural elements include task description, context, examples, output format, constraints, and other components.

**中文:** 结构元素包括任务描述、上下文、示例、输出格式、约束条件等组成部分。

**Key Concepts / 核心概念:**
- Task description / 任务描述
- Context section / 上下文部分
- Output format / 输出格式

**Example / 示例:**
```
[INSTRUCTIONS]
...
[CONTEXT]
...
[EXAMPLES]
...
[OUTPUT]
→ Well-organized prompt structure

```
