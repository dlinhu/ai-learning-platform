### 1. Ollama 基础操作 / Ollama Basic Operations

**English:** Learn basic operations for pulling, running, and managing models with Ollama.

**中文:** 学习Ollama的模型拉取、运行和管理基础操作。

**Key Concepts / 核心概念:**
- Model pulling / 模型拉取
- Model running / 模型运行
- Model management / 模型管理
- Local inference / 本地推理

**Example / 示例:**
```python
# Pull model
ollama pull llama3.2
# Run model
ollama run llama3.2
# → Basic Ollama model operations

```
