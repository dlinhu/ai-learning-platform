# Streamlit 聊天机器人 / Streamlit Chatbot

## Overview / 概述

Build interactive chatbot UIs using Streamlit for AI agents. / 使用Streamlit为AI Agent构建交互式聊天机器人UI。

## Key Knowledge Points / 核心知识点

### 1. Streamlit UI 开发 / Streamlit UI Development

**English:** Use Streamlit to quickly build agent chat interfaces with message history and session management.

**中文:** 使用Streamlit快速构建Agent聊天界面，支持消息历史和会话管理。

**Key Concepts / 核心概念:**
- Streamlit / Streamlit框架
- Chat interface / 聊天界面
- Session state / 会话状态
- Message history / 消息历史

**Example / 示例:**
```python
import streamlit as st
st.chat_message('user').write(prompt)
st.chat_message('assistant').write(response)
# → Create chat interface with Streamlit

```
