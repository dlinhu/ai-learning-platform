# Graph RAG / 图RAG

## Overview / 概述

Learn Graph RAG that uses knowledge graphs to enhance retrieval and reasoning capabilities. / 学习图RAG，使用知识图谱增强检索和推理能力。

## Key Knowledge Points / 核心知识点

### 1. Knowledge Graph Integration / 知识图谱集成

**English:** Graph RAG combines knowledge graphs with vector retrieval, providing structured knowledge and semantic understanding.

**中文:** Graph RAG将知识图谱与向量检索结合，提供结构化知识和语义理解。

**Key Concepts / 核心概念:**
- Entity relationships / 实体关系
- Structured knowledge / 结构化知识
- Graph traversal / 图遍历

**Example / 示例:**
```python
1. Extract entities from query
2. Find related nodes in graph
3. Traverse relationships
4. Combine with vector retrieval
5. Generate comprehensive answer
# → Combine graph and vector retrieval

```

---

### 2. Multi-hop Reasoning / 多跳推理

**English:** Enable multi-hop reasoning through knowledge graph relationship chains for questions requiring connected knowledge.

**中文:** 通过知识图谱的关系链实现多跳推理，回答需要连接多个知识点的问题。

**Key Concepts / 核心概念:**
- Relationship chains / 关系链
- Reasoning paths / 推理路径
- Knowledge connection / 知识连接

**Example / 示例:**
```python
Query: "Who is the CEO of the company that acquired Instagram?"
Hop 1: Instagram → acquired by → Meta
Hop 2: Meta → CEO → Mark Zuckerberg
# → Two-hop reasoning through knowledge graph

```
