# 课程列表日期标签优化计划

## 一、需求概述

按照课程编排文档中的每日规划，给每个课程列表添加醒目的日期标签，让用户能清晰看到学习进度安排。

## 二、现状分析

### 当前日期显示
- `Lesson.date` 字段存储日期（如"3月16日"）
- 前端显示为普通灰色文本，不够醒目
- 缺少日历图标和视觉层次

### 课程编排文档日期格式
- 格式：`3月16日`、`4月3日`、`5月6日`等
- 特殊标记：`⏰` 表示时间警告，`🔥` 表示高阶主题

## 三、技术方案

### 3.1 前端展示优化

**日期标签设计：**
- 使用日历图标 + 日期文本
- 蓝色背景标签，醒目显示
- 显示星期几（可选）

**展示效果：**
```
┌─────────────────────────────────────────────────────┐
│ [1] 📅 3月16日  原子 + 零/少样本                     │
│     📚 原子 · 分子 · 最小Prompt                      │
└─────────────────────────────────────────────────────┘
```

### 3.2 样式设计

```tsx
// 日期标签样式
<span className="inline-flex items-center px-2 py-0.5 rounded text-sm font-medium bg-blue-100 text-blue-700 dark:bg-blue-900/30 dark:text-blue-400">
  📅 {lesson.date}
</span>
```

## 四、实施步骤

### 修改 ModuleDetail.tsx

1. 将日期从普通文本改为醒目的标签样式
2. 添加日历图标（📅）
3. 调整布局，让日期更突出
